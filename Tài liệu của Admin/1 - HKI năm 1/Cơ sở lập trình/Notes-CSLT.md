## Chi tiết linh tinh

Môn này lớp lý thuyết không có điểm danh, không có thi lý thuyết GK tập trung vì thầy gộp điểm BT trên lớp và thi GK làm chung.

Các file thực hành chặn download, mình dùng tool tải nên mới lấy được.

Thi TH CK khó, dài (tuy đã lưu file bài làm của mình nhưng đã quên nội dung đề thi) và phải code trăm dòng trong thời gian ngắn, bonus thêm quả ai chạy cũng báo lỗi trong Visual Studio (lỗi bộ nhớ ẩn do dùng pointer sai cách). Tuy nói code ai chạy k đc auto 0 nh sau đó thầy đã chấm code lấy điểm.
